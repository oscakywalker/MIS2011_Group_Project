# Summary Report

Summary Report  
截至2026年4月26日，基于2022年7月至2026年5月12日的结构化数据，企业舆情与情感风险分析显示整体情绪格局呈现积极态势，但存在阶段性波动。从情感比例来看，当前最新月份（2026年5月）的正面情绪占比为79.52%，中性情绪为18.07%，负面情绪为2.41%。这一数据表明，尽管整体情绪偏积极，但负面情绪仍需关注，尤其是在近期出现小幅上升趋势。  

在时间维度上，情感趋势呈现出明显的阶段性波动。例如，在2023年中期至2024年初，负面情绪占比曾达到峰值，约为5.8%；而在2025年中后期，负面情绪下降至约2.5%。这种波动可能与市场事件、政策调整或品牌活动相关。此外，未来预测数据显示，到2026年9月，不同模型对情感分布的预测结果存在差异，其中RNN模型预测正面情绪为68.58%，中性情绪为28.82%，负面情绪为7.57%；而GradientBoosting模型则预测正面情绪为70.14%，中性情绪为26.97%，负面情绪为3.81%。这反映出模型之间在预测终点的情感分布上存在一定分歧，但总体趋势趋于一致，即正面情绪占主导地位。  

从数据源贡献来看，微博（Weibo）和抖音（Douyin）是主要的数据来源，分别贡献了1254条和1238条记录，合计占比超过58%。其他平台如小红书（Rednote）和大众点评（Dianping）也提供了重要数据支持，分别为822条和1097条。这些数据源的分布反映了用户在不同平台上的活跃度及信息传播特点。  

在风险特征方面，负面情绪主要集中在“愤怒”（Angry）和“悲伤”（Sad）两类情感，分别占总情感样本的26.76%和5.33%。这提示企业在应对舆情时应重点关注与愤怒情绪相关的事件，以减少潜在的声誉风险。  

模型预测的一致性方面，RNN模型在准确性、F1分数和精确度上表现最佳，其准确率为96.59%，F1分数为0.9495，精确度为0.9385。相比之下，ANOVA模型的准确率为93.43%，虽然低于RNN，但在预测终点的负面情绪占比上最低，仅为4.86%。这表明不同模型在预测终点的情感分布上存在一定分歧，但整体趋势一致。  

未来到预测终点（2026年9月），各模型预测的正面情绪占比均高于当前水平，且负面情绪占比普遍低于当前值。这表明，随着数据积累和模型优化，情感风险有望进一步降低。  

针对上述分析，建议管理层采取以下行动：  
1. 加强对微博和抖音等高活跃度平台的舆情监控，特别是涉及愤怒情绪的内容，及时响应并干预潜在风险。  
2. 优化负面情绪处理机制，建立快速反应流程，以应对突发舆情事件，防止负面情绪扩散。  
3. 持续评估不同模型的预测效果，结合多模型预测结果，提高情感风险预测的准确性和可靠性。  
4. 增加对小红书和大众点评等平台的数据采集力度，以获取更全面的用户反馈信息，提升舆情分析的覆盖范围。  
5. 定期开展情感风险评估报告，结合历史数据与预测结果，制定长期舆情管理策略，确保品牌形象稳定发展。
